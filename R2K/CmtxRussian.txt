 R2K-Commands.txt.01005-/text
 \ ROOT FILE    / \ID./
                       |
                     Ch.
 The 'Ch.' part can be:
 '' (none): This is a parameter.
 '-': The name. This follows the CMDB format rules, *including the notes on prefixes*.
      Hence, to use the ordinary format syntax, use @@ as a prefix.
      To translate "Move$!", you should probably convert it to something like "@@Move{A: #A to #B}".
      After translating this file, take as many games as possible and run the transcripter on them.
 '.': The description.
